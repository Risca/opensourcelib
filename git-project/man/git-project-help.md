git-project-help(1) -- Help with finding help
================================

## SYNOPSIS

`git-project-help` [-g,--guides] [-h] [--help] [--web] [command]

## DESCRIPTION

Shows available git project commands, or help about a specific one.

## OPTIONS

  * -g,--guides:
    List available guides

  * -h:
    Show short usage description and exit

  * --help:
    Show this man page and exit
    
  * --web:
    Display manual page for the command or guide in html format, in a browser.

## DETAILS
git-project-help performs:

  * git help git-project-command [--web]
